# BOM & Loadout Manager

This package is set up for a private GitHub repository.

## Files
- `index.html` = the app
- `bom-data.json` = all W# contents and loadout data

## First setup in GitHub
1. Open your private repo.
2. Upload `index.html` and `bom-data.json`.
3. Commit the files.
4. Open `index.html` from the repo, or download the repo and run it locally.

## Normal workflow
1. Open the app.
2. Add or edit a W#.
3. Click **Export Updated bom-data.json**.
4. In GitHub, replace the old `bom-data.json` with the exported one.
5. Commit changes.

## Notes
- The app reads `bom-data.json`.
- Your edits save in the browser while you work.
- GitHub is updated only when you export and replace `bom-data.json` in the repo.
- If you keep the repo private and do not publish Pages, other people cannot see the repo contents unless you give them access.
